## Getting Started

Git clone

```bash
$ git clone https://github.com/foxpage/foxpage-documentation.git
```

Install dependencies,

```bash
$ npm i or yarn
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Deploy

```bash
node server/app-express.js
```
